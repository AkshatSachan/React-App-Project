# Simple-Notes
<p align="center"> A simple note taking application.That allows single click creation and deletion of notes.</p><br>
<p align="center"><img src="./proj-preview.gif"></p>
<hr>
<h3 align="center">Steps to run the project on local server:</h3><br>
<p align="center">
1. run <code>npm install</code> <br>
2. run <code>npm start</code> <br>
3. run <code>npm run electron</code><br>
The desktop application should be running now.<br>
</p>
